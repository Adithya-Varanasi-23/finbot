# FinAI: Personal Finance Assistant 💸🤖

An AI-powered personal finance web application built using Streamlit and the Google Gemini API. It features an intelligent chatbot, financial calculators, an expense analyzer (via CSV), and a document summarizer (via PDF).

## Features

- **💰 Dashboard**: A quick overview of your financial health score based on the 50/30/20 budgeting rule.
- **💬 AI Chatbot**: A ChatGPT-style interface powered by Gemini 1.5 Flash to ask any finance-related questions.
- **🧮 Calculators**: Interactive EMI, SIP Growth, and Fixed Deposit calculators with Plotly visualizations.
- **📊 Expense Analyzer**: Upload your bank CSV statements, automatically categorize transactions, and get an AI-generated spending analysis.
- **📄 Document Summarizer**: Extract text from financial PDFs and ask Gemini to highlight the key insights.
- **🎨 Modern Fintech UI**: Custom sleek dark mode CSS.

## Local Setup

### 1. Prerequisites
- Python 3.9+
- A Google Gemini API Key. Get one for free at [Google AI Studio](https://aistudio.google.com/).

### 2. Installation
Clone this repository or download the source code, then install the dependencies:
```bash
pip install -r requirements.txt
```

### 3. Environment Variables
Create a file named `.env` in the root directory (you can copy `.env.example`) and add your API key:
```env
GEMINI_API_KEY=your_actual_api_key_here
```

### 4. Run the Application
Start the Streamlit development server:
```bash
streamlit run app.py
```

Open `http://localhost:8501` in your browser.

## Deployment Guide (GitHub + Streamlit Community Cloud)

To share this app publicly, the easiest and free way is to use Streamlit Community Cloud.

### Step 1: Upload to GitHub
1. Create a new repository on your [GitHub account](https://github.com/new).
2. Do not initialize with a README (keep it entirely empty).
3. Open your terminal in the `personal-finance-assistant` folder and run:
   ```bash
   git init
   git add .
   git commit -m "Initial commit of FinAI"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
   git push -u origin main
   ```
   *(Replacing `YOUR_USERNAME` and `YOUR_REPOSITORY_NAME` appropriately)*

### Step 2: Deploy on Streamlit Cloud
1. Go to [share.streamlit.io](https://share.streamlit.io/) and log in with your GitHub account.
2. Click **"New app"**.
3. Select the repository you just created in Step 1.
4. Set the **Main file path** to `app.py`.
5. Click on **"Advanced settings..."** before deploying!
6. Under **Secrets**, paste your API key exactly like this:
   ```toml
   GEMINI_API_KEY="your_actual_api_key_here"
   ```
   *(This ensures your API key remains hidden from the public repo)*
7. Click **Save**, then hit **Deploy!**

In a few minutes, your AI Personal Finance Assistant will be live on the web!
